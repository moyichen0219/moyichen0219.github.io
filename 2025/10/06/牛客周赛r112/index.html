A题 ICPC Regionals

我们先来整理一下题意和注意事项。

题目要点整理

已知条件：

  • 总队伍数 n (n >= 200 && n <= 500)
  • 小苯队伍的排名 k (k >= 1 && k <= n)
  • 金奖数量 = n * 0.1 后向上取整
  • 银奖数量 = 金奖数量 × 2
  • 铜奖数量 = 金奖数量 × 3

思考：

  • 第 1 名 ~ 第 金奖数量 名：金奖（"Gold Medal"）
  • 第 (金奖数量 + 1) 名 ~ 第 (金奖数量 + 银奖数量) 名：银奖（"Silver Medal"）
  • 第 (金奖数量 + 银奖数量 + 1) 名 ~ 第 (金奖数量 + 银奖数量 + 铜奖数量) 名：铜奖（"Bronze Medal"）
  • 其余：未获奖（"Da Tie"）

注意：得奖人数不是整数时要求向上取整

B题 Card Game

我们先来整理一下题意和注意事项。

题目要点整理

已知条件：

  • 初始有 n 张牌，点数为 a_i 。
  • 可以执行任意次操作：
    • 选择 k 张牌弃掉 (k >= 1 && k <= 当前牌总数)
    • 获得一张点数为 k 的新牌。
  • 目标是让 点数最大的牌的点数 尽可能大。
  • 最后输出这个最大值。

思考：

  • 操作的本质：弃掉 k 张牌，得到一张点数为 k 的牌。
    • 新牌的点数 = 弃掉的牌数。
  • 因此可以把问题化为 n 与 a_max 的大小比较。
    • 若 a_max 小于 n ，那么我们可以选择放弃所有牌（即n张牌）使 点数最大的牌的点数 尽可能大。
    • 反之，a_max为点数最大的牌。
