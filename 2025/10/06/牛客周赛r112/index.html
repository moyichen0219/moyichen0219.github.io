第一场牛客周赛小记

A题

我们先来整理一下题意和注意事项。

题目要点整理

已知条件：

  • 总队伍数 n (n >= 200 && n <= 500)
  • 小苯队伍的排名 k (k >= 1 && k <= n)
  • 金奖数量 = n * 0.1 后向上取整
  • 银奖数量 = 金奖数量 × 2
  • 铜奖数量 = 金奖数量 × 3

奖项划分：

  • 第 1 名 ~ 第 金奖数量 名：金奖（“Gold Medal”）
  • 第 (金奖数量 + 1) 名 ~ 第 (金奖数量 + 银奖数量) 名：银奖（“Silver Medal”）
  • 第 (金奖数量 + 银奖数量 + 1) 名 ~ 第 (金奖数量 + 银奖数量 + 铜奖数量) 名：铜奖（“Bronze Medal”）
  • 其余：未获奖（“Da Tie”）

注意：得奖人数不是整数时要求向上取整
